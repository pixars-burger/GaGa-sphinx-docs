Basic Concepts
==============

本章节将介绍C/C++的基本概念和语法。

数据类型
--------

C/C++支持多种数据类型，包括基本类型和用户定义类型。

基本数据类型
~~~~~~~~~~~~~

* int
* float
* double
* char

用户定义类型
~~~~~~~~~~~~~

* struct
* union
* enum

控制结构
--------

C/C++提供了丰富的控制结构来控制程序的执行流程。

条件语句
~~~~~~~~

* if
* else
* switch

循环语句
~~~~~~~~

* for
* while
* do-while
