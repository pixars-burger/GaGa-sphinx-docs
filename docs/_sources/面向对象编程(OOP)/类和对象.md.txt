### 类和对象
##### 1. 类和对象的基本概念

- **类（Class）**：类是定义对象属性和行为的蓝图或模板。它由数据成员（属性）和成员函数（方法）组成。
- **对象（Object）**：对象是类的实例，通过类创建。
##### 2. 定义类和创建对象

以下是一个简单的类定义和对象创建的示例：
```cpp
#include <iostream>

// 定义一个名为Person的类
class Person {
public:  // 公有访问控制
    // 数据成员
    std::string name;
    int age;

    // 成员函数
    void introduce() {
        std::cout << "Hello, my name is " << name << " and I am " << age << " years old." << std::endl;
    }
};

int main() {
    // 创建Person类的对象
    Person person1;

    // 设置对象的属性
    person1.name = "Alice";
    person1.age = 30;

    // 调用对象的方法
    person1.introduce();

    return 0;
}

```
