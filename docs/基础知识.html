基础知识¶

介绍¶

C++是一种通用的编程语言，广泛用于系统编程、游戏开发、嵌入式系统和高性能计算等领域。它由Bjarne Stroustrup在20世纪80年代开发，是C语言的扩展，增加了面向对象编程的特性。

C++的主要特点¶

  1. 面向对象编程：支持类和对象，封装、继承和多态性。

  2. 高性能：由于与硬件的紧密联系，C++可以编写高效的程序。

  3. 标准库：包括丰富的函数库，支持数据结构和算法的实现。

  4. 可移植性：在不同的平台上编写和运行程序。

基本语法¶

C++的基本语法构成了编写程序的基础。以下是一些关键概念和示例

头文件引入

#include <iostream>  // 引入标准输入输出流库

主函数

int main() {
    std::cout << "Hello, World!" << std::endl;  // 输出 "Hello, World!"
    return 0;  // 返回值，0 表示程序成功结束
}

注释

// 单行注释
/*
 * 多行注释
 */

变量声明

int a = 5;  // 整数变量
double b = 3.14;  // 双精度浮点变量
char c = 'A';  // 字符变量

基本输入输出

#include <iostream>

int main() {
    int num;
    std::cout << "Enter a number: ";  // 提示用户输入
    std::cin >> num;  // 获取用户输入
    std::cout << "You entered: " << num << std::endl;  // 输出用户输入
    return 0;
}
